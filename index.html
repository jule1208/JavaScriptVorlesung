Kurzanleitung für Canvas API

Wozu braucht es diese WEB-API?

Die Canvas-API bietet ein Mittel zum Zeichnen von Grafiken über JavaScript
und das HTML canvas Element.
Es kann unter anderem für Animationen, Spielgrafiken, Datenvisualisierung, Fotomanipulation und Echtzeit-Videoverarbeitung verwendet werden.
Die Canvas-API konzentriert sich weitgehend auf 2D-Grafiken.
Die WebGL-API, die auch das canvas Element verwendet, zeichnet hardwarebeschleunigte 2D- und 3D-Grafiken.

Sie greifen auf ein canvas Element mit der HTML-DOM-Methode getElementById() zu. Um im Canvas zu zeichnen, muss außerdem ein 2D-Kontextobjekt erstellt werden.

Notiz:

Das canvas Element selbst verfügt über keine Zeichenfunktionen.
Deshalb muss JavaScript verwendet werden, um Grafiken zu zeichnen.
Die getContext()Methode gibt ein Objekt mit Werkzeugen (Methoden)
zum Zeichnen zurück.

Syntax:

context.fillStyle = "Farbe";

context.fillRect (x, y, Breite, Höhe)


Ein grundlegendes Beispiel:

Dieses einfache Beispiel zeichnet ein hellblaues Rechteck auf eine Leinwand.


    const canvas = document.getElementById("canvas");
    const ctx = canvas.getContext("2d");

    ctx.fillStyle = "lightblue";
    ctx.fillRect(10, 10, 150, 100);
